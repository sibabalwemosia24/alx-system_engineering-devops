Shell basics
pwd- prints absolute path name of current directory
ls- display contents list of current directory
cd- change current directory to hom directory
ls -l- display current directory contents in log form
ls -l -a- display current directory contents in long form including hidden files
mkdir /tmp/my_first_directory- create my_first_directory directoy in tmp directory
mv /tmp/betty /tmp/my_first_directory- move file from derectory tmp to directory my_first_directory
rm -r tmp/my_first_directory/betty- to delete file betty from my_first_directory directory
rm -r- delete directory
cd ..- go to previous working directory
ls -la . .. /boot- list all files in current dir, parent dir and /boot dir
